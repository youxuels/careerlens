# CareerLens 🔍 职业透视镜

> **这才是高校就业服务应有的样子。**

**🚀 在线体验**: [https://youxuels.github.io/careerlens/](https://youxuels.github.io/careerlens/)

## 🎯 这是什么

CareerLens 是一个**基于学生背景 + 市场数据 + AI推理**的职业路径预测系统。

**输入**你的学校、专业、技能、软技能 → **输出**岗位匹配、Offer预测、学长对标、提升路径。

**五层产品逻辑：**

```
输入 → 匹配 → 预测 → 对标 → 路径
```

| 层级 | 做什么 | 举例 |
|------|--------|------|
| 🔍 输入 | 学校 + 专业 + 技能 + 软技能自评 | 南京大学·计算机·基础编程+AI工具·分析思维8/10 |
| 🎯 匹配 | 基于知识图谱匹配岗位 | 软件工程师 匹配度85% · AI应用工程师 匹配度78% |
| 💰 预测 | 预估Offer范围 + 具体企业 | 字节跳动 28-35万 · 华为 25-38万 |
| 👥 对标 | 同背景学长学姐真实去向 | "学长A→字节跳动 年薪32万" |
| 🛤️ 路径 | 量化能力提升建议 | "学会LLM应用开发→薪资+30%" |

## 🔥 为什么高校就业处需要这个

### 现在的就业服务太落后了

高校就业处还在用**三板斧**：

1. ❌ **改简历** — 格式改来改去，内容千篇一律
2. ❌ **发招聘信息** — 群发JD，学生看了也不知道自己够不够格
3. ❌ **办双选会** — 排队投简历，效率极低

**学生根本不知道自己的定位，也不知道差距在哪。**

### 如果每个大学都有 CareerLens

- 📊 **大一**就能看到四年后的可能性 — "以我现在的背景，毕业大概能拿多少？"
- 🎯 **大二**就知道该补什么技能 — "学会数据分析，匹配度从45%→75%"
- 🏃 **大三**就知道该冲哪些企业 — "冲刺字节跳动，保底美团"
- 🎓 **毕业前**就心中有数 — 不再海投，精准出击

**这才是就业服务应有的样子。**

## ✨ 功能特性

- 🏫 **南京本科院校全覆盖** — 25所高校，5级分层加权
- 📚 **10大专业大类 + 80+具体专业** — 两级联动选择
- 💻 **宽泛技能标签** — 基础编程/数据分析/AI工具使用等，不设门槛
- 🧠 **5维软技能自评** — 分析性思维、韧性/适应性、领导力、创造性思维、动机/自我认知
- 🤖 **AI推理引擎** — 可接入DeepSeek/GPT等LLM做深度分析
- 🇺🇸 **美国政府职位参照** — 基于USAJobs API和Open Skills API
- 📊 **4维数据可视化** — 软技能雷达图、薪资分布、行业匹配、技能价值
- 🍎 **Apple风格UI** — 浅色、大圆角、丝滑动效

## 🚀 快速开始

### 纯前端，零部署

```bash
# 克隆仓库
git clone https://github.com/youxuels/careerlens.git
cd careerlens

# 直接打开
open index.html

# 或用本地服务器
python -m http.server 8765
# 访问 http://localhost:8765
```

### AI推理（可选）

在页面的「AI推理配置」中填入：
- API地址（如 `https://api.deepseek.com/v1/chat/completions`）
- API Key
- 模型名称（如 `deepseek-chat`）

不配置也能用，会使用内置知识库。

## 🏗️ 技术架构

```
CareerLens
├── index.html          # 单文件应用，零依赖部署
├── 内置知识库
│   ├── 10大专业大类职业图谱
│   ├── 25所南京高校分层权重
│   ├── 80+具体专业映射
│   ├── 5维软技能权重矩阵
│   └── USAJobs美国政府职位数据
├── AI推理引擎
│   ├── 本地知识库匹配算法
│   └── LLM API深度分析（可选）
└── 可视化引擎
    └── Chart.js 4.4
```

**技术栈：** HTML5 + CSS3 + Vanilla JS + Chart.js

**设计原则：** 单文件、零构建、零部署、开箱即用
